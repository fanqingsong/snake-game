# 贪吃蛇游戏使用指南

## 快速开始

### 方法一：直接打开
在文件管理器中找到 `snake.html` 文件，双击用默认浏览器打开。

### 方法二：使用本地服务器
```bash
# 启动HTTP服务器
python3 -m http.server 8080

# 在浏览器中访问
http://localhost:8080/snake.html
```

## 游戏控制

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` | 控制蛇的移动方向（首次按键开始游戏） |
| `空格键` | 暂停/继续游戏 |
| `重新开始` 按钮 | 重置游戏 |

## 游戏规则

1. **开始游戏**: 按任意方向键启动游戏
2. **移动目标**: 控制蛇吃到红色的食物（圆点）
3. **得分**: 每吃一个食物得1分，蛇身增长一节
4. **失败条件**:
   - 蛇头撞到墙壁
   - 蛇头撞到自己的身体
5. **难度递增**: 每得5分，游戏速度加快

## 技术特性

- **纯前端实现**: HTML + Canvas + JavaScript
- **本地存储**: 最高分自动保存到浏览器
- **响应式设计**: 美观的渐变UI界面
- **无需安装**: 直接在浏览器中运行

## 系统要求

- 任意现代浏览器（Chrome、Firefox、Edge、Safari等）
- 支持HTML5 Canvas

## 自定义修改

如需调整游戏参数，编辑 `snake.html` 中的以下变量：

```javascript
const gridSize = 20;        // 每格大小
const tileCount = 20;       // 网格数量
let gameSpeed = 100;        // 初始速度（毫秒）
```

## 常见问题

**Q: 游戏无法开始？**
A: 按任意方向键即可开始游戏。

**Q: 如何暂停？**
A: 按空格键暂停/继续。

**Q: 最高分会保存吗？**
A: 是的，最高分保存在浏览器本地存储中。

**Q: 游戏速度可以调整吗？**
A: 游戏会根据得分自动加速，也可以修改代码中的 `gameSpeed` 变量。
